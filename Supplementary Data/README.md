# Data Repository — Optimization of PURE system composition using automation and active learning

Bernard-Lapeyre, Cleij, Sakai, Huguet, Danelon

This repository contains the experimental data behind the manuscript. Two active learning campaigns are documented:

1. `meYFP_optimization/` — single-gene reporter optimization (mEYFP, co-expressed with lacZ). Figures 1–4, S1, S2, S4.
2. `MSG1.1_optimization/` — 41-kb multicistronic synthetic chromosome optimization (mVenus/mCherry reporters). Figures 5–8, S2 (second instance).

Each subfolder has its own README with campaign-specific detail. This file covers the overall figure map and naming conventions shared across both.

## Naming conventions

| File pattern | Meaning |
|---|---|
| `Concentrations_X.csv` | METIS-generated table of component concentrations (relative to REF) tested in round X |
| `Volumes_X.csv` | Echo-compatible transfer volumes derived from `Concentrations_X.csv` |
| `Results_X.csv` | `Concentrations_X.csv` + measured relative yield (the file METIS learns from) |
| `Round_X_fluorescence_kinetics.xlsx` | Raw mEYFP/mVenus/mCherry fluorescence time-course for round X |
| `Round_X_meYFP_concentration.xlsx` / `*_concentration.xlsx` | Fluorescence converted to µg/mL using the calibration curve (Fig. S2) |
| `Round_X_absorbance_kinetics.xlsx` | CPRG→CPR absorbance time-course (lacZ / β-gal assay) |
| `Kinetic_parameters.csv` / `*_kinetic_parameters.csv` | Fitted yield + apparent translation rate per condition |
| `statistics_summary.xlsx` | Mean, SD, CV% across technical replicates |
| `*Echo_Cherry_Picking.csv` | List of "most informative" prior conditions re-tested in a later round |
| `Destination_Plate_X.xlsx` | Plate-layout map for round X |
| `All_*_merged.csv` / `All_Results_*.csv` | Concatenated, de-duplicated dataset across all rounds of a campaign — feeds the ML / kymograph / MDS scripts |
| `MS_*.csv` / `MS_Round_X_Raw.xlsx` | LC-MS proteomics abundances |
